<h1>Front-end Project Prod</h1>



<h3>Конфигурация</h3>

Полная конфигурация проекта с нуля (Webpack). 
 - React, 
 - Typescript,
 - Babel, 
 - scss, 
 - css modules, 
 - vite

<h3>UI</h3>

 - Модальные окна, выпадающие списки\меню, сайдбар, 
   кнопки с разными темами, скелетоны, попапы, ленивые изображения, аватары и т.д


<h3>Архитектура - Feature-Sliced Design</h3>

- Архитектура. Модули. Декомпозиция. Бизнес сущности. Переиспользование.



<p>Вся работа с данными осуществляться с помощью Redux toolkit и используется фейк бекенд</p>
